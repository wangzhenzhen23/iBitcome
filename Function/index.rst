钱包管理
----------

在主界面（首页）中有如下功能

创建钱包 

导入钱包 将其他钱包的私钥/助记词/keystore导入iBitcome

切换钱包 多钱包管理

钱包更名 本地个性化钱包名称

密码设置 更改钱包密码

导出私钥 将钱包（单地址）私钥/keystore导出备份
